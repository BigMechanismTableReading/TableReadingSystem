README = Extract Write

-Package Structure-
This package contains the index card and index card writer classes, which are used for outputting the extracted information

IndexCard-
Index Card stores all the information that is needed for the index cards.
Puts information into the format that the index card needs.

IndexCardWriter-
IndexCardWriter takes an index card, a table,a start and stop time.
Writes the information to a json file following the index card format.
